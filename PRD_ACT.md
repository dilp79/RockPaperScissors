# Камень-Ножницы-Бумага: Шестиугольная Стратегия

## Исходное техническое задание (ТЗ)

```
дано поле в виде шестиугольников, размера 20 на 20	
пошаговая игра, делает ход компьютер, потом игрок	
компьютер случайным образом выбирает камень, ножницы или бумагу и ставит их на поле в случайные свободные клетки. и повторяет 3 раза.
	
игрок выбирает карточку камень, ножницы или бумага и выбирает куда её поставить на поле в свободную клетку	
поле проверяется на предмет возможности очистки	
	если в клетке находится камень, в соседних с ним нет бумаги, а есть ножницы, то ножницы помечаются к удалению с поля
	если в клетке находится бумага, в соседних с ним нет ножниц, а есть камень, то камень помечается к удалению с поля
	если в клетке находится ножницы, в соседних с ним нет камня, а есть бумага, то бумага помечается к удалению с поля
после проверки всего поля все помеченные к удалению клетки очищаются.	
	за каждую очщенную клетку игрок получает 1 очко.
ход переходит к компьютеру	
игра заканчивается когда нет сводобных клеток (проверка делается до и после хода компьютера)

потребуется 3 картинки - stone, scissors, paper, они уже в папке.

при установке на поле любого предмета играй звук button_click.mp3
при очистке поля играй звук victory.mp3
```

## Обновленное ТЗ и реализованные улучшения

### Изменения в игровом поле
- Размер поля уменьшен с 20x20 до 10x10 для более комфортной игры и лучшей производительности
- Улучшена визуализация шестиугольной сетки с более точными размерами и выравниванием
- Размер шестиугольников увеличен для лучшей видимости элементов
- Точно настроены отступы между шестиугольниками для идеальной сетки

  - При наборе 20 очков компьютер размещает 4 предмета за ход
  - При наборе 40 очков компьютер размещает 5 предметов за ход
  - При наборе 60 очков компьютер размещает 6 предметов за ход
- Добавлен стартовый экран для активации звуков в соответствии с требованиями браузеров
- Добавлена визуальная индикация текущей сложности

### Улучшения интерфейса

### Ключевые компоненты

#### 1. Шестиугольная сетка
Реализована с использованием CSS-свойства `clip-path` для создания шестиугольников. Ряды смещены относительно друг друга для достижения эффекта шестиугольной сетки.

```css
.hexagon {
    width: 52px;
    height: 60px;
    background-color: #e0e0e0;
    clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0% 75%, 0% 25%);
}

.hexagon-row:nth-child(even) {
    margin-left: 26.5px;
}
```

#### 4. Алгоритм определения соседей

Для шестиугольной сетки соседние клетки определяются с учетом четности строки:

```javascript
function getNeighbors(row, col) {
    const isEvenRow = row % 2 === 0;
    
    let neighbors = [
        { row: row - 1, col: isEvenRow ? col - 1 : col }, // Верхняя левая
        { row: row - 1, col: isEvenRow ? col : col + 1 }, // Верхняя правая
        { row: row, col: col - 1 },                       // Левая
        { row: row, col: col + 1 },                       // Правая
        { row: row + 1, col: isEvenRow ? col - 1 : col }, // Нижняя левая
        { row: row + 1, col: isEvenRow ? col : col + 1 }  // Нижняя правая
    ];
    
    return neighbors.filter(n => 
        n.row >= 0 && n.row < GRID_SIZE && 
        n.col >= 0 && n.col < GRID_SIZE
    );
}
```
